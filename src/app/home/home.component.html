Get it done

Get It Done is a TODO-application created by Ferre Donné, a student at Thomas More in Geel. Get It Done is designed to be a simple but yet very effictive solution for a TODO-application.

Get started

Manage your projects easily

Get It Done allows you to manage your projects easily. You can assign deadlines, create subtasks and much more!

Anywhere, Anytime

You can use Get It Done anywhere. As long as you have your phone with you, you'll have access to all of your projects.

Color Code

With Get It Done you can easilty color code your todos anyway you want.
